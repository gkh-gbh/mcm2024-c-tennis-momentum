# Data Analysis Workflow

``` mermaid
flowchart TD
    A["明确题意与输出指标<br/>(要预测/优化/评估什么?)"] --> B["数据获取<br/>题目数据 + 外部数据"]
    B --> C["数据审计/可信度检查<br/>来源权威性/时间范围/单位/缺失率"]
    C --> D["数据清洗<br/>重复/格式/单位统一/编码修正"]
    D --> E{"缺失值处理"}
    E -->|少量缺失| E1["删除/简单填补<br/>均值/中位数/众数"]
    E -->|结构性缺失| E2["规则填补<br/>业务约束/插值/模型填补"]
    E1 --> F{"异常值处理"}
    E2 --> F
    F -->|测量错误| F1["修正/截断/剔除"]
    F -->|真实极端| F2["保留并做稳健建模<br/>RobustScaler/分位数变换"]
    F1 --> G["探索性分析EDA<br/>分布/相关性/分组对比"]
    F2 --> G
    G --> H["特征工程<br/>派生特征/滞后项/比值/人均/地理距离等"]
    H --> I{"特征选择/降维"}
    I --> I1["过滤式<br/>chi2/ANOVA/互信息"]
    I --> I2["包裹式<br/>RFE/逐步回归"]
    I --> I3["嵌入式<br/>L1/树重要性/Permutation"]
    I --> I4["降维<br/>PCA/因子分析"]
    I --> J["建模准备<br/>Pipeline/标准化/编码"]
    J --> K["划分与验证<br/>CV/时间序列切分/留出集"]
    K --> L["模型训练与对比<br/>基线->改进->集成"]
    L --> M["解释与敏感性分析<br/>可解释性/稳健性/情景模拟"]
    M --> N["结果可视化与报告<br/>图表/表格/可复现说明"]
```
